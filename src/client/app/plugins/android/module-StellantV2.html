Methods

createButton(name, x, y, sx, sy, functionName, display)

createButton method allows a definitiion of a device button, as well as his associated function that should be executed at the button pressing.
Parameters:
Name Type Description
name String the button name.
x Int the button horizontral translation.
y Int the button vertical translation.
sx Int the button horizontal size.
sy Int the button vertical size.
functionName String defines the function to be executed when the button is pressed.
display bool informs for UI changes after button pressing.

createDisplay(name, x, y, sx, sy, start_text, color, font, display)

createDisplay method allows a definitiion of a device display where can be represented any kind of text.
Parameters:
Name Type Description
name String the display name.
x Int the display horizontral translation.
y Int the display vertical translation.
sx Int the display horizontal size.
sy Int the display vertical size.
start_text String defines the text presented on the display when device started.
color String defines the color of the display text. Should be defined as hexadecimal color.
font String defines the font of evice text.
display bool defines display visibility.

createImage(name, x, y, visible)

createImage method allows a definitiion of a image or set of images that will be part of the user interface. In the Android app images will be bisposed in layers and at the bottom layer will be the fist image defined and at the top layer will be the last.
Parameters:
Name Type Description
name String the image name.
x Int the image horizontral translation.
y Int the image vertical translation.
visible bool defines image visibility.

createImage(name, x, y, visible)

addImage method allows a definitiion of a image or set of images that will be part of the user interface, after a pressing in some defined button. In the Android app images will be bisposed in layers and at the bottom layer will be the fist image defined and at the top layer will be the last.
Parameters:
Name Type Description
name String the image name.
x Int the image horizontral translation.
y Int the image vertical translation.
visible bool defines image visibility.

createJava()

createJava method allows the creation of JAVA and XML files, that will be used to code thw Android app. This files can be oppened as an Android Studio project to be modified as pleased by the user.
