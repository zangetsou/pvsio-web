NSTRUCTIONS FOR APP CREATION WITH JAVASCRIPT FRAMEWORK


1. Build your JS script, following given tutorials on tutorials/module-StellantV2.hmtl and tutorials/tutorial-StellantV2.hmtl.
2. Extract Template.zip
3. Open Terminal or Command Line on this folder and type node <your-filename>.js (NOTE: NodeJS is required to be installed).
4. Place images used by app in Template/app/src/main/res/drawable and application icons into Template/app/src/main/res/mimap-* folders. Also place needed fonts in Template/app/src/main/assets/fonts
5. Open Android Studio. Go to File->New->Import Project... -> Select extracted 'Template' folder.
6. Using Android Studio you can now edit everything your want. Don't forget to finish State.java functions.
